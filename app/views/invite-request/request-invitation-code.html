Request an invitation code

To protect you from fraud we need to send an invitation code by post to your home address.

You can request a code if you:

  • will reach State Pension age in the next 3 months
  • have not received your code or have lost it

You should receive your invitation code within 5 days of requesting it. You can then return to GOV.UK to get your State Pension.
